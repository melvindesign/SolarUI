# Introduction

**SolarUI** is an open-source UI component library designed to help you build beautiful, accessible, and consistent interfaces faster.

Built on top of [shadcn/ui](https://ui.shadcn.com/), [Tailwind CSS](https://tailwindcss.com/), and [Radix UI](https://www.radix-ui.com/), SolarUI provides a set of copy-paste-ready components that you own and can customize to fit your design system.

## Philosophy

SolarUI follows the philosophy of **ownership over dependency**. Instead of shipping a black-box component package, SolarUI gives you the source code directly in your project — you own it, you style it, you adapt it.

- **Accessible** — Components are built on Radix UI primitives, ensuring ARIA compliance out of the box.
- **Composable** — A small set of primitive components you can mix and match.
- **Themeable** — Full support for light/dark mode using CSS variables.
- **TypeScript-first** — Every component is typed.
