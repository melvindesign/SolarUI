# Theming

SolarUI uses **CSS variables** for theming, making it easy to customize the look and feel without touching component code. All colors, radii, and font settings are defined as CSS custom properties.

## CSS Variables

By default, your `globals.css` includes a set of CSS variables for both light and dark modes:

```css
@layer base {
  :root {
    --background: 0 0% 100%;
    --foreground: 222.2 84% 4.9%;

    --card: 0 0% 100%;
    --card-foreground: 222.2 84% 4.9%;

    --popover: 0 0% 100%;
    --popover-foreground: 222.2 84% 4.9%;

    --primary: 222.2 47.4% 11.2%;
    --primary-foreground: 210 40% 98%;

    --secondary: 210 40% 96.1%;
    --secondary-foreground: 222.2 47.4% 11.2%;

    --muted: 210 40% 96.1%;
    --muted-foreground: 215.4 16.3% 46.9%;

    --accent: 210 40% 96.1%;
    --accent-foreground: 222.2 47.4% 11.2%;

    --destructive: 0 84.2% 60.2%;
    --destructive-foreground: 210 40% 98%;

    --border: 214.3 31.8% 91.4%;
    --input: 214.3 31.8% 91.4%;
    --ring: 222.2 84% 4.9%;

    --radius: 0.5rem;
  }

  .dark {
    --background: 222.2 84% 4.9%;
    --foreground: 210 40% 98%;
    /* ... */
  }
}
```

## Customizing Colors

To change the primary color of your project, simply update the `--primary` variable:

```css
:root {
  --primary: 24 100% 50%; /* Orange */
  --primary-foreground: 0 0% 100%;
}
```

<Callout type="info">
  CSS variables use HSL values in the format `hue saturation% lightness%` without the `hsl()` wrapper, so Tailwind utilities can apply opacity modifiers.
</Callout>

## Dark Mode

SolarUI supports dark mode via the `dark` class on the `<html>` element. To enable it, use [next-themes](https://github.com/pacocoursey/next-themes):

```bash
npm install next-themes
```

Then wrap your app in a `ThemeProvider`:

```tsx
// app/providers.tsx
'use client'

import { ThemeProvider } from 'next-themes'

export function Providers({ children }: { children: React.ReactNode }) {
  return (
    <ThemeProvider attribute="class" defaultTheme="system" enableSystem>
      {children}
    </ThemeProvider>
  )
}
```

## Border Radius

Adjust the global border radius token:

```css
:root {
  --radius: 0.75rem; /* more rounded */
}
```
