# Installation

Solar UI s'appuie sur **shadcn/ui**, **Tailwind CSS v4** et **Next.js 14+**. Les composants sont distribués via un registre shadcn hébergé localement.

<Steps>

### Créer un projet Next.js

```bash filename="terminal" copy
npx create-next-app@latest mon-projet --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --yes
cd mon-projet
```

<Callout type="info">
  npm interdit les majuscules dans les noms de packages. Utilise des minuscules (ex : `mon-projet`, pas `MonProjet`).
</Callout>

### Initialiser shadcn

```bash filename="terminal" copy
npx shadcn@latest init --defaults
```

Cette commande génère `components.json` et installe les fichiers de base (`button.tsx`, `utils.ts`, etc.).

### Ajouter le thème Solar UI

```bash filename="terminal" copy
npx shadcn@latest add http://localhost:3000/r/solar-theme.json
```

Cela crée `src/app/solar-theme.css` et configure le style `base-nova` dans `components.json`.

### Importer le thème dans globals.css

Ouvre `src/app/globals.css` et ajoute ces deux lignes **tout en haut**, avant `@import "tailwindcss"` :

```css filename="src/app/globals.css" copy
@import "./solar-theme.css";
@import "tailwindcss";
```

### Configurer le registre Solar UI

Pour que shadcn sache où résoudre les composants Solar UI, ajoute cette entrée dans `components.json` :

```json filename="components.json" copy
{
  "registries": {
    "solar": {
      "url": "http://localhost:3000/r"
    }
  }
}
```

<Callout type="info">
  Sans cette configuration, shadcn cherche les dépendances Solar UI sur le registre officiel et retourne une erreur `solar-theme.json was not found`.
</Callout>

### Ajouter des composants

```bash filename="terminal" copy
npx shadcn@latest add http://localhost:3000/r/button.json
```

Remplace `button.json` par le composant de ton choix. Les composants sont installés dans `src/components/ui/`.

### Utiliser un composant

```tsx filename="src/app/page.tsx" copy
import { Button } from '@/components/ui/button'

export default function Page() {
  return <Button>Click me</Button>
}
```

</Steps>
