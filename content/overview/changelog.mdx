### Design system

- **Radix UI color scale** вЂ” 6 color groups (`default`, `brand`, `error`, `success`, `warning`, `info`), 12 steps each, accessible contrast guaranteed
- **Dark mode** вЂ” automatic via Radix color scoping, no `dark:` prefixes needed
- **Tailwind CSS v4** вЂ” CSS-first configuration, no `tailwind.config.js` required
- **Unified control height** вЂ” all interactive controls aligned to `2.25rem`
- **CSS variable theming** вЂ” override any token to match your brand

### Infrastructure

- **shadcn-compatible registry** вЂ” install any component with `npx shadcn@latest add https://solar-ui.com/r/[component].json`
- **MCP server** вЂ” Model Context Protocol endpoint for AI assistant integration at `https://solar-ui.com/api/mcp`
- **Documentation site** вЂ” built with [Nextra](https://nextra.site/), full-text search via Pagefind
- **Vercel Analytics & Speed Insights** вЂ” performance monitoring on the docs site
