## [0.1.0] — 2026-03-11

### Added

- Initial release of SolarUI
- `Button` component with variants: `default`, `destructive`, `outline`, `secondary`, `ghost`, `link`
- `Button` size variants: `default`, `sm`, `lg`, `icon`
- Tailwind CSS v4 integration
- shadcn/ui integration
- CSS variable-based theming with dark mode support
- Documentation site powered by [Nextra](https://nextra.site/)
- Foundation pages: Colors, Typography
