### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state of the switch. |
| `defaultChecked` | `boolean` | `false` | Initial checked state (uncontrolled). |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when the checked state changes. |
| `disabled` | `boolean` | `false` | When `true`, prevents interaction and applies reduced opacity. |
| `required` | `boolean` | `false` | Makes the switch required in a form context. |
