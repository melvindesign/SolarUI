### Manual

**1. Install the following dependency:**

```bash filename="terminal" copy
npm install @radix-ui/react-switch
```

**2. Copy and paste the following code into your project:**

```tsx filename="components/ui/switch.tsx" copy
"use client"

import * as React from "react"
import * as SwitchPrimitive from "@radix-ui/react-switch"
import { cn } from "@/lib/utils"

// Root: h-5 w-9 rounded-full, checked=bg-primary, unchecked=bg-input
// Thumb: size-4 bg-background, translate-x-0 -> translate-x-4
```

### Switch

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | — | Controlled checked state of the switch. |
| `defaultChecked` | `boolean` | `false` | Initial checked state (uncontrolled). |
| `onCheckedChange` | `(checked: boolean) => void` | — | Callback when the checked state changes. |
| `disabled` | `boolean` | `false` | When `true`, prevents interaction and applies reduced opacity. |
| `required` | `boolean` | `false` | Makes the switch required in a form context. |
| `size` | `"default" \| "compact"` | `"default"` | Controls the size of the switch. |
