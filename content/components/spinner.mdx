### Manual

No additional dependencies are required. Copy and paste the following code into your project:

```tsx filename="components/ui/spinner.tsx" copy
import { cn } from "@/lib/utils"
import { Loader2Icon } from "lucide-react"

function Spinner({ className, ...props }: React.ComponentProps<"svg">) {
  return (
    <Loader2Icon role="status" aria-label="Loading" className={cn("size-4 animate-spin", className)} {...props} />
  )
}

export { Spinner }
```

### Spinner

| Prop        | Type                                         | Default     | Description                                 |
| ----------- | -------------------------------------------- | ----------- | ------------------------------------------- |
| `size`      | `"sm" \| "default" \| "lg" \| "xl"`         | `"default"` | Controls the size of the spinner.           |
| `className` | `string`                                     | —           | Additional CSS classes to apply.            |

All other props are forwarded to the underlying `<span>` element. The component includes a visually hidden `Loading…` label for screen readers via `aria-label` and an `sr-only` span.
