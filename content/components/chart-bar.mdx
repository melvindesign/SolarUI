# Bar Chart

A chart that displays data as vertical or horizontal bars, ideal for comparing values across categories.

<Tabs.Tab>

```tsx
"use client"

import { Bar, BarChart, CartesianGrid, XAxis } from "recharts"
import {
  ChartContainer,
  ChartTooltip,
  ChartTooltipContent,
} from "@/components/ui/chart"
```

```tsx
const chartData = [
  { month: "Jan", desktop: 186, mobile: 80 },
  { month: "Feb", desktop: 305, mobile: 200 },
]

const chartConfig = {
  desktop: { label: "Desktop", color: "var(--chart-1)" },
  mobile: { label: "Mobile", color: "var(--chart-2)" },
}

<ChartContainer config={chartConfig} className="h-64 w-full">
  <BarChart data={chartData}>
    <CartesianGrid vertical={false} />
    <XAxis dataKey="month" />
    <ChartTooltip content={<ChartTooltipContent />} />
    <Bar dataKey="desktop" fill="var(--color-desktop)" radius={4} />
    <Bar dataKey="mobile" fill="var(--color-mobile)" radius={4} />
  </BarChart>
</ChartContainer>
```

### Manual

**1. Install the dependency:**

```bash filename="terminal" copy
npm install recharts
```

**2. Copy `components/ui/chart.tsx` into your project** (shared `ChartContainer`, `ChartTooltip`, `ChartLegend` primitives).

### ChartContainer

| Prop | Type | Description |
|------|------|-------------|
| `config` | `ChartConfig` | Maps data keys to labels and colors. |
| `className` | `string` | Set height with `h-*` (required). |

### ChartConfig

```ts
type ChartConfig = {
  [key: string]: {
    label?: string
    color?: string       // CSS variable or hex
    theme?: { light: string; dark: string }
  }
}
```

Colors are injected as `--color-{key}` CSS variables and should be used as `fill="var(--color-desktop)"`.
