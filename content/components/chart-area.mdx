# Area Chart

A line chart with a filled area beneath, useful for showing cumulative totals or volume over time.

<ComponentPreview>
  <AreaChartDefault />
</ComponentPreview>

<Tabs items={['Usage', 'Installation', 'API']}>

<Tabs.Tab>

```tsx
"use client"

import { Area, AreaChart, CartesianGrid, XAxis } from "recharts"
import {
  ChartContainer,
  ChartTooltip,
  ChartTooltipContent,
} from "@/components/ui/chart"
```

```tsx
<ChartContainer config={chartConfig} className="h-64 w-full">
  <AreaChart data={chartData}>
    <CartesianGrid vertical={false} />
    <XAxis dataKey="month" />
    <ChartTooltip content={<ChartTooltipContent />} />
    <Area
      dataKey="desktop"
      fill="var(--color-desktop)"
      fillOpacity={0.2}
      stroke="var(--color-desktop)"
      strokeWidth={2}
    />
  </AreaChart>
</ChartContainer>
```

### Manual

**1. Install the dependency:**

```bash filename="terminal" copy
npm install recharts
```

**2. Copy `components/ui/chart.tsx` into your project.**

<Tabs.Tab>

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `string` | `"linear"` | Interpolation curve type. |
| `fillOpacity` | `number` | `0.6` | Transparency of the filled area. |
| `stackId` | `string` | — | Share the same id across areas to stack them. |
| `strokeWidth` | `number` | `1` | Stroke thickness. |
