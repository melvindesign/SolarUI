# Radial Chart

A circular bar chart that radiates from a center point, ideal for showing progress or part-to-whole relationships.

<ComponentPreview>
  <RadialChartDefault />
</ComponentPreview>

<Tabs items={['Usage', 'Installation', 'API']}>

<Tabs.Tab>

```tsx
"use client"

import { RadialBar, RadialBarChart } from "recharts"
import {
  ChartContainer,
  ChartTooltip,
  ChartTooltipContent,
} from "@/components/ui/chart"
```

```tsx
<ChartContainer config={chartConfig} className="h-72 w-full">
  <RadialBarChart data={chartData} innerRadius={30} outerRadius={110}>
    <ChartTooltip content={<ChartTooltipContent />} />
    <RadialBar dataKey="desktop" fill="var(--color-desktop)" background cornerRadius={4} />
  </RadialBarChart>
</ChartContainer>
```

<Tabs.Tab>

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `innerRadius` | `number` | `0` | Radius of the inner hole. |
| `outerRadius` | `number` | — | Maximum bar radius. |
| `startAngle` | `number` | `90` | Start angle in degrees. |
| `endAngle` | `number` | `-270` | End angle. Full circle when difference is 360. |
| `background` | `boolean` | `false` | Show a background track behind each bar. |
| `cornerRadius` | `number` | `0` | Rounded bar corners. |
