# Icons

SolarUI uses **Phosphor Icons** by default, with Lucide as a fallback. Click any icon to copy its component name to the clipboard.

```tsx
import { icons } from "@/components/ui/icons"

const { IconArrowLeft, IconCheck } = icons
```
